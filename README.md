# TORCH_AIHUB  üî•

TorchAIHub is a collection of deep learning projects built with [PyTorch](https://pytorch.org/), dedicated to computer vision.  
It serves as a hub for experiments and production-ready implementations, designed to be clear, maintainable, and scalable.

---

##  Goals
- Showcase practical computer vision applications using PyTorch
- Provide modular, reusable code for learning
- Document best practices for training, evaluation, and deployment
- Serve as a portfolio of deep learning projects

---


##  Getting Started
Clone the repo and install dependencies:
```bash
git clone https://github.com/AyoubArdem/TORCH_AIHUB.git
pip install -r requirements.txt
```

---

##  Requirements
- Python 3.9+
- PyTorch (latest stable release)
- TorchVision
- Additional dependencies listed in `requirements.txt`
